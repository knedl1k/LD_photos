# LinuxDays_photos
A simple BASH script to download photos from LinuxDays2023.

LinuxDays website: https://www.linuxdays.cz/2023/

## How to use
```
git clone https://github.com/knedl1k/LinuxDays_photos
cd LinuxDays_photos
bash download.sh
```
